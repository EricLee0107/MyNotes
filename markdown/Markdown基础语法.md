# 基础语法



## 标题



Markdown只支持6个级别的标题，级别越高标题越小。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
####### 七级标题
```

## 引用

在文本前加入大于号 > 即可

**代码格式：**

```markdown
> 这是一个引用
```

**具体效果**

> 这是一个引用

引用里面还可以再包含引用，可以用于做引用分块

例如：

```markdown
> 这是一个引用
>> 这是另一个引用
```

> 这是一个引用
>
> > 这是另一个引用

## 列表

Markdown支持有序列表和无序列表两种形式

- 无序列表使用`*`或者`+`或者`-`标识
- 有序列表使用数字加`.`标识，例如：`1.`



### 无序列表

```markdown
* 星号无序列表1
* 星号无序列表2
* 星号无序列表3

+ 加号无序列表1
+ 加号无序列表2
+ 加号无序列表3

- 减号无序列表1
- 减号无序列表2
- 减号无序列表3
```

**具体效果如下：**

* 星号无序列表1
* 星号无序列表2
* 星号无序列表3



+ 加号无序列表1
+ 加号无序列表2
+ 加号无序列表3



- 减号无序列表1
- 减号无序列表2
- 减号无序列表3



### 有序列表

```markdown
1. 有序列表1
2. 有序列表2
3. 有序列表3
	1. 2222
```

**具体效果如下：**

1. 有序列表1

2. 有序列表2

3. 有序列表3



### 任务列表

```markdown
- [ ] 一个待办任务
- [X] 一个已完成任务
```







### 使用列表的一些注意事项

如果在单一列表项中包含了多个段落，为了保证渲染正常，*与段落首字母之间必须保留四个空格

```markdown
*    段落一

     小段一
*    段落二

     小段二
```

以上标记显示效果如下：

- 段落一

  小段一

- 段落二

  小段二

另外，如果在列表中加入了区块引用，区域引用标记符也需要缩进4个空格

```markdown
* 段落一
    > 区块标记一
* 段落二
    > 区块标记二
```

- 段落一 

  > 区块标记一

- 段落二 

  > 区块标记二

注：记住一个原则，如果在和列表配合使用的时候出现了问题，就缩进一次，四个空格或者一个制表符代表一次缩进。如果一次缩进没有解决问题，那就两次。

## 常用的文字编辑操作

### 粗体与斜体

粗体语法：想要粗体的文字前后各输入两个星号 如：`**`粗体字`**`

斜体语法：想要斜体的文字前后个输入一个星号

**语法代码：**

```markdown
**粗体字**
*斜体字*
```

**具体效果：**

**粗体字**
*斜体字*

### 删除字（划线删除）

语法：在文字的前后分别加两个`~`则代表这段文字被划线。

```markdown
~~被划线删除的字~~
```



**具体效果：**

~~这是一条删除线~~





## 分割线

语法：在空白行下方添加三条“-”横线

**语法举例：**

```markdown
---
```

**具体效果：**

---



# 高级语法

## 插入代码块

### 单行代码

语法：
单行代码：代码之间分别用一个反引号包起来，单行代码不需要输入语言标识。

```markdown
    `代码内容`
```

**语法举例：**

```markdown
`单行代码内容;`
```

**具体效果：**

`单行代码内容;`

### 多行代码

就像上面语法举例一样，显示的时某种语言的代码格式（markdown也是一种编码语言），在代码块中，根据对应的编码语言，部分内容会有高亮显示。

语法格式是：三个<font color=red>\`</font>(键盘坐上角数字键1左边的那个键)，然后可以添加对应的语言标识符，然后换行再三个<font color=red>\`</font>

**语法举例：**

```markdown
​``` go
var tt int32
​```
```

**具体效果：**

``` go
var tt int32
```



## 插入表格

语法：

```
表头|表头|表头
---|:--:|---:
内容|内容|内容
内容|内容|内容

第二行分割表头和内容,并且标识内容的表格当前列的对齐方式
```

对齐方式：

- `——:`右对齐

- `:——`左对齐

- `:——:`居中对齐

- `——` 默认对齐方式（居中）

示例：

```
班级|姓名|学号
--|:--:|--:
1|小明|1001
2|李华|1002
3|Bob|3001
```

| 班级 | 姓名 | 学号 |
| ---- | :--: | ---: |
| 1    | 小明 | 1001 |
| 2    | 李华 | 1002 |
| 3    | Bob  | 3001 |
